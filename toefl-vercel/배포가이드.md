```

```

---

## 🚀 배포 순서

### 1단계 — GitHub 저장소 만들기
1. https://github.com → New repository
2. 이름 입력 (예: `toefl-practice`) → Create repository
3. 이 폴더 전체를 업로드:
   ```bash
   git init
   git add .
   git commit -m "first commit"
   git remote add origin https://github.com/내아이디/toefl-practice.git
   git push -u origin main
   ```

### 2단계 — Vercel 연결
1. https://vercel.com → Sign up (GitHub 계정으로 로그인)
2. **Add New Project** → GitHub 저장소 선택 (`toefl-practice`)
3. Framework: **Other** 선택
4. **Deploy** 클릭

### 3단계 — API Key 환경변수 등록 (중요!)
1. Vercel 대시보드 → 해당 프로젝트 → **Settings** → **Environment Variables**
2. 아래 값 추가:
   - Name: `ANTHROPIC_API_KEY`
   - Value: `sk-ant-api03-...` (실제 API 키)
   - Environment: Production, Preview, Development 모두 체크
3. **Save** 클릭
4. **Deployments** 탭 → 최신 배포 → **Redeploy** (환경변수 적용을 위해)

### 4단계 — 접속 확인
- Vercel이 자동으로 `https://toefl-practice-xxx.vercel.app` 주소 제공
- 커스텀 도메인 연결도 가능 (Settings → Domains)

## ✅ 배포 후 동작 방식

```
사용자 브라우저
    ↓ (API Key 없이 요청)
/api/chat  (Vercel 서버)
    ↓ (서버에서 API Key 자동 첨부)
Anthropic API
```

- 사용자는 API Key 입력 불필요
- API Key는 서버 환경변수에만 존재 → 완전히 안전

## ❓ 자주 묻는 오류

| 오류 | 원인 | 해결 |
|------|------|------|
| `API key not configured` | 환경변수 미설정 | 3단계 재확인 후 Redeploy |
| `404 /api/chat` | vercel.json 경로 오류 | vercel.json 확인 |
| 빈 화면 | public/index.html 경로 | 폴더 구조 확인 |
